Do's & Don'ts

DO

Use #F7F4EF as page background — never pure white
Use Basque Crimson Red only for hero, CTAs, active nav, scores
Render eyebrow labels uppercase with +1.5px letter-spacing
Use #E5DED6 as universal border on all light surfaces
Switch text to Deep Ink once below 300px hero gradient zone
Reserve weight 900 for Display hero title only
Give championship content amber treatment
Ensure 44×44px minimum touch targets

DON'T

Don't use #FFFFFF as page background
Don't use pure black #000000 — use Deep Ink #141414
Don't place dark ink text over the red hero gradient
Don't add box-shadows to standard list cards
Don't use corner-rounding below 6px on interactive elements
Don't compress section spacing below 80px
Don't use more than one accent color per page section
Don't render critical content client-side only
